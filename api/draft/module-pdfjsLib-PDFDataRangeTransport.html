Class: PDFDataRangeTransport

pdfjsLib~PDFDataRangeTransport(length, initialData, progressiveDoneopt, contentDispositionFilenameopt)

Abstract class to support range requests file loading. NOTE: The TypedArrays passed to the constructor and relevant methods below will generally be transferred to the worker-thread. This will help reduce main-thread memory usage, however it will take ownership of the TypedArrays.
